# TIB OS v3.0 — The Indie Tech Investor Operating System

> **思想开源，数据收费。**
>
> 投资的价值在于深度思考和逻辑博弈，而非重复性的数据录入。

**TIB OS v3.0** 是勇麦将**十年投资经验与思考"代码化"**的 AI 投资辅助系统。它将复杂的投资决策流程拆解为 **15 个核心投资元技能 (Prompts)**，并整合到一个**开箱即用的前端交互界面**中。

旨在帮助独立投资者：告别繁琐的手动输入，实现更理性、更高效、更系统化的投资决策。

## 🧠 核心理念：AI 不替你投资，AI 帮你少犯错

我们对 AI 在投资中的角色有清晰的定义——

| AI 应该做的 | AI 不应该做的 |
|---|---|
| **补充视角**：帮你看到自己思维盲区之外的信息 | ❌ 替你预测涨跌 |
| **遵循纪律**：在你情绪上头时强制执行投资规则 | ❌ 替你做最终决策 |
| **减少情绪犯错**：用系统隔离冲动、FOMO、恐慌 | ❌ 保证投资收益 |
| **结构化拆解**：把模糊的盘感变成可复用的逻辑链 | ❌ 取代独立思考 |

**最终的买卖判断，永远由人来执行。**

AI 是你的副驾驶，不是自动驾驶。TIB OS 的价值不在于给你答案，而在于帮你**问对问题、守住纪律、看清全局**。

### 🆕 `prompt_references/` — 提示词资产文件夹

这是 v3.0 的重要更新。之前所有提示词只存在于前端页面中，现在每个核心投资元技能都有了**独立的 Markdown 文件**。

这意味着：

- **🤖 Agent 可直接调用**：把文件夹下载到本地，配置到你的 AI Agent（如 Claude、GPT、Cursor 等）的知识库中，让 AI 在投资决策时直接引用这些结构化提示词
- **🔧 可作为你自己的起点**：Fork 后根据自己的投资思路修改每个 Prompt，逐步梳理出属于你自己的"投资思路代码化"策略
- **📦 可组合调用**：不同场景调用不同 Prompt——扫描时用 P1，纠结时用 P3，持仓管理用 P4，形成你自己的投资 SOP

## 💎 核心特性

- **16 个核心投资元技能**：涵盖从市场扫描、深度调研、交易执行到持仓管理与复盘的投资全流程
- **独立 Prompt 资产**：每个提示词单独成文件，方便 Agent 调用、版本管理和个人定制
- **开箱即用的交互界面**：一个独立的 `index.html` 文件，双击即可在浏览器中运行，无需任何环境配置
- **智能变量配置**：自动识别 Prompt 模板中的变量，生成直观的输入框，告别手动修改 Prompt 的繁琐
- **本地缓存记忆**：利用浏览器 `localStorage` 自动保存用户输入，下次打开页面数据不丢失
- **自定义模板与编辑**：支持在浏览器中直接编辑和保存 Prompt 模板
- **轻量化与可扩展性**：纯 HTML/CSS/JavaScript 实现，不依赖外部库

### 方式二：下载 Prompt 资产给你的 AI Agent

```bash
# 克隆整个项目
git clone https://github.com/Evan-XYZ/tib-os-3.0.git

# 或者只下载 prompt_references 文件夹
# 将其导入到你的 AI Agent 知识库中
```

将 `prompt_references/` 文件夹导入到你使用的 AI 工具的知识库或上下文中，即可让 AI 在对话中调用这些投资元技能。

### 方式三：梳理成你自己的投资策略

1. Fork 本项目
2. 打开 `prompt_references/` 中的各个 Markdown 文件
3. 根据你自己的投资风格、风险偏好和关注领域修改每个 Prompt
4. 逐步形成属于你的"投资思路代码化"资产
